Área do Retângulo
Área do Retângulo

Feito no começo de fevereiro de 2022.

Desenhe um retângulo na tela (clique, arraste e solte) e o site lê as dimensões de um retângulo (base e altura), calcula sua área e escreve na tela.

Feito usando HTML 5, CSS 3, JavaScript e Canvas. O site está responsivo e funciona com o touch em dispositivos móveis.

A atividade pedia apenas para escrever um algoritmo para ler as dimensões de um retângulo (base e altura), calcular e escrever a área do retângulo, mas quis incrementar para que o usuário pudesse desenhar o próprio retângulo.

Obs.: não aceita quadrado!

Valeu a Pena Comprar pelo Jogo?
Valeu a pena comprar o jogo?

Feito no final de janeiro de 2022.

Descubra o ROI (Return Over Investment) sobre cada jogo que possuo.

O ROI é calculado da seguinte forma: (horas * 10) / preço = ROI, isto considerando que cada hora vale R$ 10,00 (valor aproximado ao que eu recebia por hora em meu último emprego), o que significa que cada hora me rendeu R$ 10,00 de felicidade!

Sendo assim, se eu pagar R$ 100,00 num jogo, mas só jogar ele por 4 horas, significa que meu ROI é de 40%, porque o jogo me rendeu apenas R$ 40,00 de felicidade.

Servidor para Contato
Servidor para Contato

Servidor em NodeJS usando ExpressJS que conecta no Banco de Dados Não Relacional PostgreSQL através do cliente pg, que tem a implementação no Heroku e as queries realizadas usando Knex.

O formulário de contato deste portfólio dispara os dados preenchidos para este servidor, que protege os dados com senha. Ou seja, só eu posso consultar os dados.

Como o servidor geralmente não tem uma interface visual, deixo a imagem comparando Front e Back End só para ilustrar...
